Instrucciones
Realizar los siguientes ejercicios detal manera que se presenten en una pagina web haciendo uso de funciones Java Script. y presentarlos en un repositorio de git hub de manera individual.

1 - Has hecho una compra y sabes el precio del producto y su iva. Haz un script que te calcule el precio total que vas a pagar por tu compra.

2 - En una variable tienes el lado de un cuadrado, debes escribir un programa que te calcule el área y el perímetro del cuadrado.

3 - En este script debes pedir al usuario un número de dos dígitos y debes devolver el número de unidades y de decenas, o sea, cada dígito del número. Usa solo operaciones aritméticas.

4 - Una tortilla de patatas lleva 200 gramos de patatas por persona. Por cada kilo de patatas se necesitan 5 huevos y 300 gramos de cebolla. Escribe un script que dado el número de comensales calcule las cantidades de ingredientes necesarias.

5 - Juego de adivinar el numero entre 0 y 100, al final debe mostrar el numero de intentos y los números ingresados, el usuario tiene un máximo de 15 intentos los cuales se deben mostrar en cuenta regresiva; adicional dar pistas si el numero ingresado es mayor o menor al que se debe adivinar.

6 - Serie Fibonacci donde el usuario decide el numero de secuencias a mostrar.

7 - Realizar programa; Introducción de 2 números y el ordenador indica cual es
el mayor o si son iguales.

8- Realizar programa que calcule el factorial de un número entero no negativo ingresado por el usuario.

9 - Escribe un programa que solicite al usuario dos números y luego muestre todos los números múltiplos de 6 que hay entre esos dos números, incluyendo los extremos.

10 - Crear un programa que genere un arreglo con 20 números entre 0 y 100, muestre este arreglo y luego cree 2 arreglos adiciones donde se separan pares de impares.

11 - Crear un programa que genere un arreglo con 10 números entre 10 y 20, lo muestre y luego elimine los múltiplos 3 mostrando como quedaría el arreglo modificado.

12 - dados los 3 lados de un triangulo estimar su área, y determinar si es equilátero, isósceles o escaleno.

13 - Crear un arreglo con n palabras, generar un nuevo arreglo con la cantidad de caracteres de cada palabra y por ultimo determinar de cuantos caracteres seria la oración con estas palabras.

14 - Generar un arreglo con 10 números, generar 2 nuevos arreglos, con uno invertir su contenido de manera algorítmica y el otro con el uso de la función (reverse)

15 -  Realizar calculadora básica, donde el usuario decide la operación a realizar mediante input type radio
